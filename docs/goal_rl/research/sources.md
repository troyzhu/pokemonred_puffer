# Sources — bibliography

Numbered bibliography for the goal_rl design notes. Cite by ID
(`[S##]`) from other pages.

## Source tiers

- **Tier A** — peer-reviewed journal/conference papers (preferred).
  Cite author + venue + year.
- **Tier B** — verified technical reports from reputable labs
  (DeepMind, OpenAI, Anthropic) with named authors.
- **Tier C** — arXiv from a clearly named lab/institution, flagged as
  such. Design inspiration only, not evidence.
- **Excluded** — Medium, uncurated blogs, unverifiable arXiv IDs,
  Wikipedia as primary.

`[verify]` marks entries where venue/year/authorship still needs
confirmation.
